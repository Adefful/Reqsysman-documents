## Финальный вариант 
```mermaid
%%{init: {'theme':'dark'}}%%
erDiagram
    RequirementType ||--|{ Requirement : "type of"
    RequirementType {
	    int id PK
        varchar title
        text description
    }
	Requirement {
		varchar id PK
		int type_id FK
		text description 
		varchar priority
		varchar status
		varchar version
	}
    Relationship {
	    int id PK
	    varchar version 
	    varchar last_hash_commit
	    varchar source_id FK
	    varchar target_id FK
    }
    Relationship }o--|| Requirement : "source"
    Relationship }o--|| Requirement : "target"
```


## Другой вариант
```mermaid
%%{init: {'theme':'dark'}}%%
erDiagram
    RequirementType ||--o{ Requirement : "type of"
    RequirementType ||--o{ RelationshipType : "source type"
    RequirementType ||--o{ RelationshipType : "target type"
    RequirementType {
	    int id PK
        varchar title
        text description
    }
	Requirement {
		varchar id PK
		int type_id FK
		text description 
		varchar priority
		varchar status
		varchar version
	}
    RelationshipType ||--o{ Relationship : "type of"
    RelationshipType {
	    int id PK
	    text description 
	    int source_type_id FK
	    int target_type_id FK
    }
    Relationship {
	    int id PK
	    varchar version 
	    varchar last_hash_commit
	    varchar source_id FK
	    varchar target_id FK
	    int type_id FK
    }
    Relationship ||--|| Requirement : "source"
    Relationship ||--|| Requirement : "target"
```
